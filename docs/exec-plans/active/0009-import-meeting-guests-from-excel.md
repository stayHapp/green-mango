# 管理员导入会议嘉宾名单

## 任务名称

在管理员会议详情中实现 Excel 嘉宾名单导入。

## 背景

已确认第一版以 Excel 作为会议嘉宾批量导入方式，嘉宾登录固定使用姓名和手机号。当前原型仅能查看 mock 嘉宾，无法验证导入流程。

## 用户确认状态

用户已确认开始执行管理员端功能，并已确定 Excel 导入和姓名加手机号登录规则。

## 目标

- 管理员可下载导入模板并选择 xlsx 文件。
- 系统导入首个工作表中的姓名、手机号及可选嘉宾信息。
- 缺少姓名或手机号的行不导入，并显示错误行数。
- 导入成功的嘉宾立即出现在当前会议嘉宾列表中。

## 不在本次范围内的内容

- 不实现真实后端文件上传、异步任务、重复手机号合并或导入历史。
- 不实现字段自由组合登录方式。

## 技术决策

- 使用已安装的 SheetJS（xlsx）在浏览器读取 Excel。
- 模板列为姓名、手机号、单位、职务、身份、座位号；姓名和手机号必填。
- 导入数据仅写入当前 mock 内存，刷新页面后恢复 JSON 初始数据。

## 涉及文件

- `frontend/src/types.ts`
- `frontend/src/mock/mockApi.ts`
- `frontend/src/views/admin/AdminMeetingDetailView.vue`
- `docs/architecture/api.md`

## 验收标准

- 管理员可下载 xlsx 模板并导入有效嘉宾。
- 无姓名或手机号的行被拒绝，页面显示结果摘要。
- `npm run build` 通过。
