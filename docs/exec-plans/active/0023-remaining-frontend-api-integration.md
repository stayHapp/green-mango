## 目标与顺序

1. 管理员 Excel 模板下载、嘉宾导入和签到表导出。
2. 管理员工作人员维护。
3. 管理员签到统计与明细。

5. 嘉宾会议、个人资料和签到二维码。

每次只完成并验证其中一个明确步骤，完成后在本文记录结果，再进入下一项。
