# 管理员端会议工作台视觉改版

## 背景

管理员端目前使用通用页面顶栏和 Element Plus 默认表格、页签样式，视觉语言与已经完成墨绿色主题的嘉宾端、工作人员端不一致。会议详情内多个管理能力集中在页签中，也不利于管理员在会议现场快速定位操作。

## 目标

- 为管理员会议列表和会议详情建立统一的桌面端管理工作台壳层。
- 以当前会议为上下文呈现侧边导航、顶部信息和会议状态。
- 将概览、会议信息、嘉宾、字段、服务、工作人员、签到记录映射为清晰的管理入口。
- 统一卡片、表格、表单、标签、按钮和空状态的墨绿色视觉风格。
- 保持现有 API、数据结构和全部业务操作不变。

## 范围

- 调整 `App.vue` 的管理员端壳层判断。
- 新增管理员工作台布局组件。
- 改造管理员会议列表和会议详情的页面结构、详情页导航及概览面板。
- 在全局样式中追加隔离的管理员主题规则。
- 更新管理员工作流和前端架构说明。

## 非范围

- 不新增或修改后端 API、数据库结构和权限模型。
- 不调整嘉宾端、工作人员端既有业务流程。
- 不更改会议、嘉宾、工作人员或签到数据。

## 验证方式

- 从管理员登录后访问会议列表，确认能创建会议和进入会议详情。
- 访问会议详情，确认侧边栏可切换概览、配置、嘉宾、字段、会议服务、工作人员和签到记录。
- 验证嘉宾新增、字段开关、会议服务编辑、工作人员维护与签到表导出入口仍可用。
- 执行前端构建、差异格式检查，并用浏览器验证桌面与窄屏布局。

## 完成结果

- 管理员会议列表和会议详情已切换到统一的会议工作台壳层。
- 会议详情已增加数据总览，并通过侧边导航切换原有管理能力。
- 后台卡片、表格、表单、标签、按钮和空状态已统一为墨绿色主题。
- 已完成前端构建、差异格式检查、真实管理员登录、会议详情导航和桌面与窄屏页面验证。
