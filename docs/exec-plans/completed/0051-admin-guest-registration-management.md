# 后台嘉宾管理与自主报名闭环调整

## 目标

将会议入口分享信息收敛到“会议信息”，完善后台嘉宾列表与字段配置，并让嘉宾自主报名能够按照字段配置提交、在后台审核后转为正式嘉宾。

## 执行范围

1. 删除后台顶部无实际功能的主题图标，并将会议入口链接和二维码仅放置在“会议信息”。
2. 为正式嘉宾记录增加来源，后台列表展示姓名、手机号、单位、职务、来源、审核状态和签到状态。
3. 让字段配置支持必填、报名表单、个人信息页和启用状态；动态字段通过公开会议接口提供给报名页。
4. 在嘉宾管理页加载并审核自主报名申请，审核通过后刷新正式嘉宾列表。
5. 更新数据库/API 文档，补充迁移和自动化测试。

## 完成与验证

- 执行后端 pytest 与前端构建。
- 在本地浏览器中验证管理员字段配置、嘉宾报名提交、后台审核和列表展示。
