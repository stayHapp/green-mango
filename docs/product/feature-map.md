# 功能地图

## 管理员端

### 会议管理

- 创建会议。
- 查看会议列表。
- 查看会议详情。
- 修改会议基本信息。
- 管理会议管理员。

### 嘉宾信息字段配置

- 为每个会议配置嘉宾信息字段。
- 字段可用于嘉宾信息展示。
- 字段可被选为嘉宾登录验证字段。
- 字段后续可支持排序、必填和展示控制。
- 第一版嘉宾登录固定使用姓名 + 手机号；字段自由组合后续支持。

### 嘉宾管理

- 提前录入嘉宾信息。
- 通过 Excel 批量导入会议嘉宾名单，并反馈错误行和导入结果。
- 查看嘉宾列表。
- 编辑嘉宾信息。
- 查看嘉宾签到状态。

### 会议服务

- 维护会议日程、会议资料、天气提醒、路线导航和联系会务。
- 分别编辑五项服务的正文、未发布提醒和发布状态。
- 配置天气和路线共用的准确导航地点。
- 第一版不允许新增、删除或调整固定入口类型。

### 工作人员管理

- 创建工作人员账户。
- 授权工作人员负责会议。
- 查看工作人员执行签到情况。

### 签到管理

- 查看签到统计。
- 查看签到明细。
- 查看未签到嘉宾。
- 导出全量嘉宾签到表。

## 嘉宾端

- 登录嘉宾端。
- 查看登录入口所属的当前会议。
- 查看当前会议相关信息和五项会议服务。
- 查看个人参会信息。
- 展示个人签到二维码。
- 签到成功后在个人二维码区域展示“已签到”标记。

## 工作人员端

- 登录工作人员端。
- 进入负责的会议。
- 扫描嘉宾二维码。
- 完成扫码签到或手工签到。
- 搜索嘉宾并核验身份、座位和签到状态。
- 在网络断开时收到提示，并在恢复网络后继续签到。
- 查看签到成功、重复签到、二维码过期、无权限等结果。

## 后续扩展功能

- 会议页面配置。
- 结构化日程管理。
- 资料上传。
- 通知能力。
- 统计分析。
- 更细粒度权限。
- 多租户支持。
- 多场次或多签到点。
- 受控撤销手工签到：仅限原操作工作人员或会议管理员在限定时间内执行，必须填写原因并保留完整操作记录。
