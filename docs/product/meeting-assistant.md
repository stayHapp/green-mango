# 会议助手

## 功能范围

会议助手沿用嘉宾端已有五个入口：会议日程、会议手册、天气情况、路线指引和联系我们。每项功能均由会议管理员维护，并按会议独立控制发布状态。

## 发布规则

- 管理员可以编辑功能正文和未发布提醒。
- 功能发布后，嘉宾点击入口查看管理员编辑的正文。
- 功能未发布时，入口仍然保留；嘉宾点击后查看管理员编辑的提醒。
- 嘉宾点击任一功能后进入独立详情页；详情页使用统一标题栏和返回图标，不在侧边抽屉内展示正文。
- 会议日程使用日期标签、纵向时间线和日程卡片呈现；其他功能沿用同一视觉体系展示内容卡片。
- 未发布功能不得向嘉宾泄露草稿正文。
- 各项功能后续可以扩展自己的结构化字段，本阶段先用通用正文验证编辑与发布闭环。

## 前端阶段边界

第一阶段使用 Mock 数据完成管理员配置和嘉宾查看。后续整体查验功能合理性后，再设计数据库结构和 API，不提前引入第三方天气、地图或文件服务。
