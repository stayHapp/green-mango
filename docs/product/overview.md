# 产品概览

## 项目背景

教育领域的会议、研讨、培训、论坛和专家讲座通常涉及活动信息发布、报名收集、参会人员管理、嘉宾与日程安排等工作。很多组织者仍依赖临时表单、群消息和手工表格协作，信息分散且难以长期维护。

知会希望为这类活动提供一个专业、可配置、可逐步扩展的管理平台。

## 目标用户

- 教育专家
- 学校领导
- 教研组织者
- 教育培训活动负责人
- 学术会议组织者

## 核心问题

1. 活动信息分散，不容易统一管理。
2. 不同会议的报名字段不同，固定表单无法适配。
3. 组织者需要快速创建会议、配置报名表并发布页面。
4. 管理员需要查看、管理和筛选报名数据。
5. 后续需要扩展会议页面配置、嘉宾管理、日程管理、资料上传、报名审核和数据导出等能力。

## 产品价值

知会不是简单的会议报名系统，而是面向教育会议和研讨活动的组织管理平台。它的核心价值是让会议活动从创建、发布、报名到后台管理具备统一结构，同时保留足够的字段配置能力。

## 项目边界

第一版只关注会议管理、报名字段配置、前台报名和后台报名记录查看。微信登录、支付、短信、邮件、复杂权限、多租户、统计分析、文件上传和高级页面装修暂不纳入 MVP。
