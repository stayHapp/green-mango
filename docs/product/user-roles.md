# 用户角色

第一版只定义最小角色，不设计复杂权限体系。

## 访问者

访问者是打开会议页面但尚未提交报名信息的人。

访问者可以：

- 查看公开会议页面。
- 查看会议基本信息。
- 查看报名表字段。

## 报名用户

报名用户是提交了某个会议报名表的人。

报名用户可以：

- 填写会议报名表。
- 提交报名信息。

第一版暂不要求报名用户登录。

## 管理员

管理员是会议组织方或平台管理人员。

管理员可以：

- 创建会议。
- 查看会议列表和会议详情。
- 修改会议基本信息。
- 配置会议报名字段。
- 查看会议报名记录。
- 查看报名用户提交的字段值。

第一版只做基础管理员能力，后续再扩展组织、角色和权限模型。
