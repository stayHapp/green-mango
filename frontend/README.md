# Frontend

知会前端项目，当前使用 Vue 3、Vite 和 TypeScript 实现管理员端、嘉宾端、工作人员端的交互原型。

计划技术栈：

- Vue 3
- Vite
- TypeScript
- Element Plus
- Vue Router
- Pinia
- Axios

## 启动开发服务

在本目录运行：

```bash
npm install
npm run dev
```

启动成功后，按终端输出的本地地址在浏览器中访问。首次运行或 `package.json`、`package-lock.json` 变更后，需要先执行 `npm install`。

## 可用命令

```bash
npm run dev      # 启动开发服务
npm run build    # 执行类型检查并构建生产产物
npm run preview  # 本地预览生产构建产物
```
